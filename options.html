#DontHostHate Extension Options


We have curated a list of "known providers"---domain registrars, hosting providers, payment providers, and other services---along with links to their Terms of Service, relevant terms, and abuse contact procedures. You can reference this list for free at any time. One optional feature of this extension will correlate this spreadsheet with the data detected on a particular website so you don't have to do it yourself.

In order to identify known providers, you must add a Google Spreadsheets API key. To use this feature, please follow Google's instructions for acquiring an API key. Note that you want an API key, not an OAuth token. This is free (and optional!). I recommend that on the page "Restrict and Rename API key" you set Application Restrictions to "HTTP referrers" and "API restrictions" to "Restrict key -> Google Sheets API".

This key will be stored in your Google Chrome profile and will only be used by you.



Whois is a tool used to identify domain registrars. An excellent online API is available from whoisxmlapi.com. You can register to get a free API key by clicking "get started free". This extension will work without the whois api, but it will not be able to identify the domain registrar of a website.

This key will be stored in your Google Chrome profile and will only be used by you.




Source code for this extension is available on github. You can also contact the developer via email or twitter.
